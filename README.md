# 🎮 Rock Paper Scissors Game (Python)

A beginner-friendly yet well-structured Python project that simulates the classic **Rock Paper Scissors** game.
This project demonstrates core programming concepts like **functions, loops, and conditional statements** in a clean and interactive way.

---

## 🚀 Features

* 🧑‍💻 User vs Computer gameplay
* 🔁 Continuous game using loops
* ⚙️ Modular code using functions
* ✅ Input validation for better user experience
* 🎯 Instant result display (Win/Loss/Draw)

---

## 🛠️ Technologies Used

* **Python**
* **Random Module**

---

## 📚 Concepts Covered

* Functions
* While Loop
* If-Else Conditions
* User Input Handling
* Basic Game Logic

---

## ▶️ How to Run the Project

1. Clone the repository:

   ```
   git clone https://github.com/your-username/rock-paper-scissors-python.git
   ```

2. Navigate to the folder:

   ```
   cd rock-paper-scissors-python
   ```

3. Run the program:

   ```
   python rock_paper_scissors.py
   ```

---

## 🎯 Sample Gameplay

```
Enter your choice (rock/paper/scissor): rock
Computer chose: paper
Result: Computer Wins 💻
```
